Technical Part 2 20/07/17

  • What is grid based design?
    • Grid-based design is a commonly used method to arrange the content on your webpage. The grid is comprised of row and column classes and preferably can be broken up into easily divisible amounts, the example in the Udacity tutorial had 12 columns as this could be divided by 2, 3 and 4. This will come in handy for display-responsive webpages when using percentages. The grid design is a great way to get your proportions looking right, offers uniformity across all your pages and gives the reader an easy to navigate webpage.

  • Whats all the hype about responsive webpages?
    • Responsive webpages give the viewer the best possible user experience of the webpage on any screen size, device or requirement. The CSS stylesheets can be customized for a number of uses for the websites, including a print-rfriendly version of the webpage it may cut down on images and focus more on the text. My favourite example of responsive webpages is when you have a webpage and manually drag and reduce the screen's resolution in the browser and the page changes to have simplified menus and it's less busy on the screen to accomodate for smart phones and tablets.

  • What is semantic structure?
    • In a way the semantic structure is what gives HTML uniformity and by using semantic tags its easier for readers and other developers to follow and understand the structure. A simple example for following semantics would be, you would give the main heading of a webpage the h1 tag and then a not so prominent heading h2 then its sub-heading h3. You wouldn't make the main heading h3 and make a small heading h1.

  • Any other points you found interesting and want to reflect on.
    • I found the CSS reset/normalize method interesting as it's meant to ensure the CSS styles to look the same across all browsers. I would like to look further into thus as I felt the Udacity tutorial only brushed over it.
