Technical Part 6 16/08/17

    Explain to a non-tech friend
    In the JavaScript language, describe all the use cases for each of:
  • parentheses () - e.g. '1. Execute a function var result = doSomething()'
    • The parentheses are used to hold expressions if it's part of a structure, or parameters if it's part of a function. When these parameters are assigned values upon being called, they are referred to as arguments.

  • brackets []
    • Square brackets are used for creating arrays and also for dereferencing them. You can also use them to dereference object properties when the dot notation isn't possible, such as string values that contain spaces or strings that start with a number.

  • braces {}
    • The curly braces are used for creating objects. Such as var myObj = {}. Another way to create an object would be var myObj = new Object().

  • single quotes '' AND double quotes ""
    • Single quotes and double quotes are interchangeable and can be useful when you want to use either set inside of the other for a string variable.
