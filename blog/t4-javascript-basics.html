Technical Part 4 04/08/17

  • How does JavaScript compare to HTML and CSS?
    • While HTML handles the structure of the webpage's content and CSS applies visual styles to these elements offering variations for different media, JavaScript adds interactive functionality and responds to events. A simple example of this is the prompt function that asks you to input something, the result can then be used for further functions or events.

  • Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.
    • JavaScript runs code from the top down but the control flow changes when it comes across structures likes conditionals and loops. An example of a conditional in everyday life would be decding what to make for dinner based on the ingredients you have. Starting with the fact that you have mince and kidney beans, if you have corn chips, make nachos, otherwise if you have rice make chili con carne, otherwise if you have neither, make mince (on toast - obviously). Now the cooking of the mince may have loop situation where you stir the mince, and check: is the mince cooked? If not keep cooking and stir, is the mince cooked? Once the mince equals cooked then the loop ends and you move onto the next process.

  • Explain the difference between accessing data from arrays and objects.
    • Arrays have a continuous index starting from 0 and data can be accessed by referencing its position in the index. Objects on the other hand is more of a collection of data made up of keys and values. To access data in objects you make reference to that specific key.

  • Explain what functions are and why they are useful.
    • Functions are great for combining several actions into a single executable action. An example of this would be if you want to do a calculation but don't want to have type the whole equation out each time (or copy and paste it - repitition is bad). Just write it out once in a function, call upon that function, input the argument values and it can output the answer in one go.
