Technical Part 31/08/17

  • Explain to a non-tech friend: What 'scope' is, and how it works in JavaScript.
    • In JavaScript scope is the term used to refer to the set of variables you have access to. This includes objects and functions, as they can be variables too. These variables can have local or global scope. An example of a variable having local scope would be a variable contained within a function where it will be used and then forgotten when it leaves that function, allowing you to use the same variable name further down the track for another function without having to worry about already using that name. A global scope variable will be defined outside of any function and can be useful if you are constantly referring back to the same variable within other functions.
